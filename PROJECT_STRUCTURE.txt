HOW TO ADD NEW CONTENT:
-----------------------
METHOD 1 - Using DialogueScreen (RECOMMENDED):
  1. Create new screen in src/components/Screens/
  2. Import DialogueScreen component
  3. Pass props: background, character, speaker, dialogue, choices
  4. See Chapter1ScreenExample.jsx for reference
  
METHOD 2 - Custom screen:
  1. Create new screen in src/components/Screens/
  2. Add route in App.jsx GameRouter
  3. Use existing components (StatsPanel, StatChangeNotification)
  4. Follow pattern from Chapter1Screen.jsx
